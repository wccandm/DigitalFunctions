# DigitalFunctions

Arduino - Library for digital functions on bools.

#### Functions:
- bool IsOn()
- bool IsOff()
- bool SR()
- bool RS()
- bool RisingPuls()
- bool FallingPuls()
- bool Toggle()

#### Input variables:
- bool input
- int debounce time (ms)
- set condition
- reset condition
